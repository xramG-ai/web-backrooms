# WEB BACKROOMS

브라우저 3D 백룸 탐색 프로젝트 — M0 (걷는 백룸)

기획서 v2의 M0 마일스톤 구현. 노란 벽지 백룸을 무한히 걸을 수 있다.

## 바로 실행
`dist/index.html`을 브라우저에서 열면 끝. (단일 파일 빌드 — 서버 불필요)

## 개발 모드
```bash
npm install
npm run dev
```

## 조작
클릭하여 진입 → WASD 이동, Shift 달리기, Space 점프, Ctrl/C 앉기, Tab 소지품, ESC 설정

## 구현된 것 (M0)
- 1인칭 컨트롤러 (포인터 락, 헤드밥, 발소리 연동)
- 시드 기반 결정론적 청크 생성 (20m 청크, 5×5 로드) — 같은 곳으로 돌아가면 같은 구조
- 청크 경계에 벽 없음 → 전역 연결 보장 (갇힘 방지)
- AABB 슬라이드 충돌 (M3에서 Rapier + 균열 설계로 교체 예정)
- 지수 안개로 유효 시야 ~10m 제한
- 절차 생성 텍스처 (노란 벽지·카펫·천장 타일 — 외부 자산 0)
- 절차 생성 오디오 (형광등 험 120/240Hz + 노이즈, 카펫 발소리 — 오디오 파일 0)
- 형광등 상태 3종: 켜짐 / 꺼짐 / 플리커
- 벽 UV 월드 크기 매핑 (텍스처 늘어남 없음), 청크당 병합 지오메트리 (드로우콜 최소화)

## 구조
```
src/
  game/
    config.ts     상수 (속도, 시야, 청크 크기, 시드)
    rng.ts        해시 + mulberry32
    chunks.ts     청크 생성 규칙 + 캐시
    collision.ts  AABB 수집 + 축 분리 슬라이드
    textures.ts   캔버스 절차 텍스처 + 재질
    audio.ts      Web Audio 험/발소리
    state.ts      프레임 전역 상태 (React 밖)
  components/
    World.tsx     청크 로드/언로드
    ChunkMesh.tsx 청크 지오메트리 병합 + 형광등
    Player.tsx    이동/시점/중력/충돌/헤드밥
  App.tsx         캔버스, 안개, 진입 오버레이
```
